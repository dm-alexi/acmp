Строки
(Время: 1 сек. Память: 16 Мб Сложность: 29%)
Циклическим сдвигом строки s называется строка s_k s_k+1 s_k+2 … s_|s| s_1 s_2 … s_k-1 для некоторого k, здесь |s| - длина строки s. Подстрокой строки s называется строка s_i s_ i+1 … s_ j-1 s_j для некоторых i и j. Вам даны две строки a и b. Выведите количество подстрок строки a, являющихся циклическими сдвигами строки b.

Входные данные
Первая строка входного файла INPUT.TXT содержит строку a (1 ≤ |a| ≤ 1000). Во второй строке входного файла записана строка b (1 ≤ |b| ≤ min(100,|a|)). Обе строки состоят только из символов английского алфавита и цифр.

Выходные данные
В выходной файл OUTPUT.TXT выведите целое число – ответ на задачу.

Примеры
№	INPUT.TXT	OUTPUT.TXT
1	abcabc		4
	abc
2	abcabc		0
	acb
3	aaaaaaa		6
	aa
4	aAaa8aaAa	4
	aAa
