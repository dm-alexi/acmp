Числа Фибоначчи
(Время: 1 сек. Память: 16 Мб Сложность: 16%)

Последовательностью Фибоначчи называется последовательность чисел a_0, a_1, ..., a_n, ..., где
a_0 = 0, a_1 = 1, a_k = a_k-1 + a_k-2 (k > 1).

Требуется найти N-е число Фибоначчи.
Входные данные

Во входном файле INPUT.TXT записано целое число N (0 <= N <= 30).
Выходные данные

В выходной файл OUTPUT.TXT выведите N-е число Фибоначчи.
Пример
№	INPUT.TXT	OUTPUT.TXT
1	7		13
